# ⚽ Brasileirão Dashboard

Dashboard interativo com dados do Campeonato Brasileiro Série A de 2006 a 2025.

## 📊 Funcionalidades

- Filtro por temporada
- Campeão, melhor ataque e melhor defesa do ano
- Tabela de classificação completa com saldo de gols
- Gráfico de pontuação por time
- Gráfico de saldo de gols
- Download da tabela em CSV

## 🚀 Como rodar

**1. Clone o repositório**
git clone https://github.com/SEU_USUARIO/brasileirao-dashboard.git

**2. Instale as dependências**
pip install -r requirements.txt

**3. Rode o dashboard**
streamlit run brasileirao_dados.py

## 🛠️ Tecnologias

- Python
- Streamlit
- Pandas
- Plotly

## 📁 Dados

Planilha com dados do Brasileirão Série A de 2006 a 2025 contendo:
posição, time, pontos, jogos, vitórias, empates, derrotas, gols marcados e sofridos.
